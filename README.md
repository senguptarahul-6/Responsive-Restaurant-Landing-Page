# Responsive Restaurant Landing Page

A modern and fully responsive restaurant landing page designed to showcase menu offerings, pricing, and customer contact details.

## Tech Stack
- HTML
- CSS
- Flexbox
- CSS Grid
- Javascript

## Features
- Fully responsive layout (mobile & desktop)
- Navigation bar with smooth scrolling
- Hero section highlighting the restaurant brand
- Pricing section for menu items
- Contact form for customer inquiries
- Clean UI with proper spacing and visual hierarchy
- Built using Flexbox and CSS Grid
